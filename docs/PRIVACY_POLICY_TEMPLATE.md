# 개인정보 처리방침

**취향 (Chuihyang)** 앱 ("회사" 또는 "우리")은 사용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다.

**최종 수정일:** 2024년 12월 12일  
**시행일:** 2024년 12월 12일

## 1. 수집하는 개인정보의 항목

### 1.1 필수 수집 정보
- **회원가입 시:**
  - 이메일 주소
  - 비밀번호 (암호화 저장)
  - 닉네임
  - 생년월일

- **서비스 이용 시:**
  - 프로젝트 정보 (담금주 이름, 레시피, 시작일, 목적, 메모)
  - 프로젝트 사진 (사용자가 업로드하는 경우)
  - 기기 정보 (OS 버전, 앱 버전, 기기 모델)
  - 알림 토큰 (푸시 알림 발송용)

### 1.2 선택적 수집 정보
- 전화번호 (프로필 정보)
- 프로필 사진

### 1.3 소셜 로그인 시 수집 정보
- **Google 로그인:**
  - 이메일 주소
  - 이름
  - 프로필 사진 (선택)

- **Apple 로그인:**
  - Apple ID (고유 식별자)
  - 이메일 주소 (선택적 제공)
  - 이름 (선택적 제공)

### 1.4 자동으로 수집되는 정보
- 앱 사용 기록 (접속 시간, 이용 기록)
- 오류 로그 (앱 안정성 개선용)
- IP 주소 (보안 및 부정 이용 방지)

## 6. 사용자의 권리

사용자는 다음의 권리를 가집니다:

### 6.1 개인정보 열람 요청

- 앱 내 프로필 화면에서 직접 수정 가능

- 앱 내 설정에서 회원 탈퇴 가능

### 6.4 동의 철회
- 알림 수신 동의는 앱 설정에서 철회 가능
- 소셜 로그인 연동은 각 플랫폼에서 해제 가능

### 6.5 데이터 이동권
- 사용자는 본인의 데이터를 요청하여 다운로드할 수 있습니다

## 7. 개인정보 보호 대책

우리는 사용자의 개인정보를 안전하게 보호하기 위해 다음의 조치를 취합니다:

### 7.1 기술적 보호 조치
- 비밀번호 암호화 저장 (bcrypt/scrypt)
- HTTPS 통신 암호화
- 데이터베이스 접근 제어 (RLS - Row Level Security)
- 정기적인 보안 업데이트

### 7.2 관리적 보호 조치
- 개인정보 접근 권한 최소화
- 정기적인 보안 교육
- 개인정보 처리 방침 정기 검토
